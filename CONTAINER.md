# Unbound Container

Container images based on [Unbound](https://nlnetlabs.nl/projects/unbound/), a validating, recursive, and caching DNS resolver with DNSSEC support.

Sources are available on [GitHub](https://github.com/anthochamp/container-unbound).

See [README.md](README.md) for full documentation.

## Image tags

- `x.y.z-unboundA.B.C`: Container image version `x.y.z` with Unbound `A.B.C`.
- `edge-unboundA.B.C`: Latest commit build with Unbound `A.B.C`.

**Tag aliases:**

- `x.y-unboundA.B.C`: Latest patch of `x.y` with Unbound `A.B.C`.
- `x-unboundA.B.C`: Latest minor+patch of `x` with Unbound `A.B.C`.
- `x.y.z-unboundA.B`: Version `x.y.z` with latest patch of Unbound `A.B` (only latest container version updated).
- `x.y-unboundA.B`: Latest patch of `x.y` with latest patch of Unbound `A.B`.
- `x-unboundA.B`: Latest minor+patch of `x` with latest patch of Unbound `A.B`.
- `x.y.z-unboundA`: Version `x.y.z` with latest minor+patch of Unbound `A` (only latest container version updated).
- `x.y-unboundA`: Latest patch of `x.y` with latest minor+patch of Unbound `A`.
- `x-unboundA`: Latest minor+patch of `x` with latest minor+patch of Unbound `A`.
- `x.y.z`: Version `x.y.z` with latest Unbound (only latest container version updated).
- `x.y`: Latest patch of `x.y` with latest Unbound.
- `x`: Latest minor+patch of `x` with latest Unbound.
- `unboundA.B.C`: Latest container with Unbound `A.B.C`.
- `unboundA.B`: Latest container with latest patch of Unbound `A.B`.
- `unboundA`: Latest container with latest minor+patch of Unbound `A`.
- `latest`: Latest `x.y.z-unboundA.B.C` tag.
- `edge-unboundA.B`: Latest commit build with latest patch of Unbound `A.B`.
- `edge-unboundA`: Latest commit build with latest minor+patch of Unbound `A`.
- `edge`: Latest `edge-unboundA.B.C` tag.
